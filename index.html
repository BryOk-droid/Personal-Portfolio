Akan Name Generator

A web application that calculates your Akan name based on your birthday and gender. Akan names are derived from Ghanaian culture, where children are given names based on the day of the week they were born. Technologies used are HTML, CSS, JavaScript. This project helps users connect with Ghanaian culture by providing a fun and interactive way to discover their Akan name. It also demonstrates my ability to build user-friendly web applications with dynamic functionality.
